# Steps to run activiti-spec docker

1. Please create a directory on host to hold configuration file.
2. Copy the provided configuration.yaml in this configuration directory and populate real enviornment value of Cassandra to which activity-spec web service needs to be connected.
3. Execute below commands to start docker containers.

# Start activity-spec init container
docker run -d --name <CONTAINER_NAME> -e CS_HOST=<>CS_HOST_IP -e CS_USER=<CS_USER> -e CS_PASSWORD=<CS_PASSWORD> <IMAGE_NAME>
docker run -d --name <CONTAINER_NAME> -e JAVA_OPTIONS="-Xmx128m -Xms128m -Xss1m" -v <CONFIG_FILE_DIRECTORY_PATH>:/etc/onap/activity-spec/be/ -p 8080:8080 <IMAGE_NAME>

# Sample Commands
docker run -d --name sdc-act-init -e CS_HOST=172.17.0.2 -e CS_USER=asdc_user -e CS_PASSWORD=Aa1234%^! onap/activity-spec-init:1.2.0-SNAPSHOT
